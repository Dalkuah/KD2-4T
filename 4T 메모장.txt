umap : 레벨 or 맵

레벨 : 게임 '월드'의 모든 것 또는 일부. 가상의 공간.
액터 : 레벨에 배치할 수 있는 오브젝트.
 - 트랜스폼이 있다. (위치, 회전, 스케일)
컴포넌트 : 액터에 추가할 수 있는 하나의 함수 기능.
 - 레벨에 단독으로 사용 불가

폴리곤 덩어리 = 메쉬
뼈대 = 스켈레탈 메쉬
 
빨강 : 이벤트
 - 특정한 시점을 호출 받는 것
파랑 : 함수
 - 어떠한 기능을 수행.
회색 : 메크로
 - 흐름을 분할하거나 컨트롤 함.

변수 : 해당 유형을 담는 그릇.

오브젝트 : 언리얼엔진에서 관리를 받는 모든 것
 - 메모리 상에 올라가 있는 (살아있는) 녀석

String과 Text의 차이점
